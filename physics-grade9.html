四、九年级核心公式速查

模块公式 / 规律使用提醒
内能Q = cmΔt注意单位：质量 kg，温度变化量 ℃。
热值Q = mq，Q = Vq固体/液体常用质量，气体常用体积。
电流I = Q / t电荷量与时间的比值。
欧姆定律I = U / R动态电路先判断电路结构，再看电表示数变化。
电功W = UIt = Pt电能表题要看转数、时间和倍率。
电功率P = UI = U²/R = I²R额定与实际问题要先区分电压状态。
焦耳定律Q = I²Rt纯电阻电路中电热等于电功。
电磁安培定则、左手定则、右手定则先定磁场方向，再判断受力或感应电流方向。

学习建议

1. 每天先做对应习题文件，再看视频订正。
2. 视频编号必须连续推进，尤其是第17章欧姆定律，跳过中间题型会影响后续动态电路。
3. 每周五完成本周错题归档：错误原因、正确模型、下次辨认信号各写一句。
4. 习题答案需要到对应课件 PPT 中查找，订正时不要只抄答案，要补全分析过程。
